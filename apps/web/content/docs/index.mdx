A comprehensive personal finance and budget management application built with Next.js 16 and React 19.

## Overview

Budget App helps you take control of your finances by providing powerful tools to:

- **Track Transactions** - Record income and expenses with categories and tags
- **Manage Budgets** - Set spending limits by category and monitor progress
- **Set Financial Goals** - Define savings targets with deadlines
- **Automate Recurring** - Handle recurring bills and income automatically
- **Get Reminders** - Never miss a bill payment again
- **Analyze Spending** - Visualize your financial data with charts and reports
